# Introduction

Ngroker is a gui client for ngrok. It is a simple and easy to use tool to create tunnels to localhost.

## What is Ngroker?

A **simple**, **powerful** and **flexible** unified ingress platform for developers with everything you love from ngrok.

## Features

- **Auto Dark Mode**: Ngroker automatically switches to a dark theme for better visibility on system prefers-color-scheme.
- **Internationalization**: Ngroker supports multiple languages, including English, Chinese, and Japanese.
- **Easy to Use**: Ngroker is designed to be easy to use, with a simple and intuitive interface.
- **Compatible for CLI**: Ngroker is compatible with the ngrok CLI, allowing users to take advantage of both the CLI and the GUI.
